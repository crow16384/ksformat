### Использование

```r
library(ksformat)

# Создать формат
sex_fmt <- format_create(
  "M" = "Male",
  "F" = "Female",
  .missing = "Unknown"
)

# Применить к данным
format_apply(c("M", "F", NA), sex_fmt)
# [1] "Male"    "Female"  "Unknown"
```
