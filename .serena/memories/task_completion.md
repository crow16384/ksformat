# Task Completion Checklist

When a coding task is completed, follow these steps:

1. **Regenerate documentation** (if any roxygen2 comments were modified):
   ```r
   devtools::document()
   ```

2. **Run tests** to verify nothing is broken:
   ```bash
   Rscript -e "devtools::test()"
   ```
   Or:
   ```bash
   Rscript -e "testthat::test_dir('tests/testthat')"
   ```

3. **Check package** (for significant changes):
   ```bash
   R CMD check .
   ```

4. **Verify NAMESPACE** is up to date (auto-generated by roxygen2):
   - New exported functions need `#' @export`
   - Internal helpers should have `#' @keywords internal`

5. **Update tests** if new functionality was added or behavior changed.

6. **Commit changes** with a descriptive message.
