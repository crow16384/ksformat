# Date/Time Format Support ('SAS'-style)

Provides 'SAS'-compatible date, time, and datetime formatting
capabilities. Built-in 'SAS' format names (DATE9., MMDDYY10., TIME8.,
DATETIME20., etc.) are auto-resolved and can be used directly with
[`fput`](fput.md), [`fputn`](fputn.md), and [`fputc`](fputc.md).
